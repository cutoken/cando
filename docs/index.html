Features

Web UI

Session management, provider switching, thinking mode, plan view, and multiple themes.

Sandboxed Execution

Shell commands, file operations, and background processes run within isolated workspaces.

Multi-Provider

Supports Z.AI and OpenRouter. Access Claude, GPT-4, Qwen, and 100+ models.

Context Memory

Auto-summarizes long conversations, stores originals in SQLite, with recall and pin tools.

Cross-Platform

Runs on Linux, macOS, and Windows. Single binary, no dependencies.

Conversation Persistence

Auto-saves sessions. Resume anytime with --resume flag.
